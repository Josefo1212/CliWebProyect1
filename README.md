# Calculadora de matrices

## Instrucciones de uso
1. Ingresar el tamaño de la matriz(entre 2 y 10).
2. Haz clic en "Crear Matriz" para generar los campos de entrada.
3. Ingresa los valores de la matriz A usando el teclado numérico en pantalla o del dispositivo y al darle al boton igual o tecla enter luego de llenar las casillas, saldra el boton de guardar la matriz.
4. Guarda la matriz A y repite el proceso para la matriz B.
5. Usa los botones de operaciones (Sumar, Restar, Multiplicar, multiplicar por escalar,transponer,determinante,inversa,identidad) para ver los resultados.

## Funcionalidades

- Crear matrices cuadradas de tamaño entre 2 y 10.
- Llenar matrices manualmente o con valores aleatorios.
- Sumar, restar y multiplicar matrices.
- Multiplicar una matriz por un escalar.
- Calcular la transpuesta de una matriz.
- Calcular el determinante de una matriz.
- Calcular la matriz inversa (si es posible calcularla).
- Generar y mostrar una matriz identidad.
- Cargar un ejemplo rapido.
- Ingresar datos por teclado fisico o teclado digital.
